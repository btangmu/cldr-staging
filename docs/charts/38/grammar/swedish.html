Grammatical Forms

Unit Inflections, Phase 1: The full goal is to add full case and gender support for formatted units. During phase 1, a limited number of locales and units of measurement are being handled in CLDR v38, so that we can work kinks out of the process before expanding to all units for all locales (those where we can get the grammatical structure).

This chart shows grammatical information available for certain units. The rows contain forms appropriate for different grammatical cases and plural forms. Each plural form has a sample value in the Plural column. That value is used with the localized unit pattern to form a formatted measure, such as “2.0 Stunden”. That formatted measure is in turn substituted into a case minimal pair phrase, which is constructed to require the particular case form. For example, “… für {0} …” requires the placeholder to be in the accusative case in German.

Notes

  • We don't have the cross-product of minimal pairs for both case and plural forms, so the case minimal pair phrase might not be correct for the row’s plural category.
  • In addition, translators often have difficulties with the the minimal pair patterns, since they are transcreations not translations.
  • It can also be difficult for translators to supply minimal pair patterns that are natural for a variety of placeholders: for example, it is probably not typical to use the vocative with 3.2 meters!
  • For clarity, conversion values are supplied for non-metric units. For more information, see Unit Conversions.
