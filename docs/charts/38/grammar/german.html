Grammatical Forms.

This chart shows grammatical information available for certain units. The rows contain forms appropriate for different grammatical cases and plural forms. Each plural form has a sample value in the Plural column. That value is used with the localized unit pattern to form a formatted measure, such as “2.0 Stunden”. That formatted measure is in turn substituted into a case minimal pair phrase, which is constructed to require the particular case form. For example, “… für {0} …” requires the placeholder to be in the accusative case in German.

Note that we don't have the cross-product of minimal pairs for both case and plural forms, so the case minimal pair phrase might not be correct for the row’s plural category.

For clarity, conversion values are supplied for non-metric units. For more information, see Unit Conversions.
